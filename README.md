# AeroStream – Real-Time Flight Telemetry Monitoring System

A system designed to process and analyze real-time flight telemetry data such as 
altitude, velocity, and position.

## Goals
- Real-time telemetry data ingestion
- Flight anomaly detection
- Continuous monitoring pipelines

## Tech Stack
Python, Data Processing

## Planned Features
- Stream processing pipeline
- Statistical anomaly detection
- Telemetry visualization dashboard
